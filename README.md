# 🛒 Simba Supermarket System

A modern and responsive supermarket web application built using **Vite + React**, with support from **Claude Code AI**. The project simulates a real-world supermarket experience with smooth UI, product browsing, and structured data handling using a **JSON-based backend**.

🔗 Live Demo: https://simba4.vercel.app

---

## 📌 Project Description

Simba Supermarket System is a frontend-focused web application designed to replicate a digital shopping experience. It allows users to browse products in a clean interface while demonstrating how modern web technologies can be combined with a lightweight backend structure.
The goal of this project is to practice building scalable UI components, manage data efficiently, and create a smooth user experience similar to real e-commerce platforms.

---

## ⚙️ Features

- 🛍️ Product listing and browsing system  
- 🔍 Simple and intuitive navigation  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- ⚡ Fast performance using Vite  
- 🧾 Data handling using JSON backend  
- 🎨 Clean and modern UI design
